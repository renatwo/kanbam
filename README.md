# Kanban Full-Stack Project

Projeto completo de Kanban construído com React, Vite, TailwindCSS (Frontend) e Node.js, Express, JWT (Backend).
Os dados são salvos em arquivos JSON de forma simples, sem necessidade de banco de dados.

## Como Executar

Você precisará de 2 abas no seu terminal:

### 1. Iniciar o Backend

Abra um terminal na pasta raiz e execute:
```bash
cd backend
npm install
npm run dev
```
*(O servidor irá rodar na porta 3001 e vai criar os arquivos JSON automaticamente)*

### 2. Iniciar o Frontend

Abra outro terminal na pasta raiz e execute:
```bash
cd frontend
npm install
npm run dev
```
*(O React abrirá na porta 5173)*

Abra http://localhost:5173 no seu navegador, crie uma conta e aproveite seu Kanban!
